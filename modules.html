cleanKey

  • cleanKey(s: string, opts?: CleanKeyOptions): string
    • Defined in index.ts:215

    Removes bad chars for a string key (all except number, lowercase ascii7 letters, dash - and underscore _) It will allow uppercase ascii7 letters if option caseSensitive is false If the mode is:

    • strict: it will allow just the standard set (0-9, a-z, -, and _)
    • dots => strict + .
    • specials => strict + ., |, ~, /, and : It also removes multiple dashes in a row and replaces them for a single dash unless option replaceManyDashes: false is given

    Parameters

    • s: string
    • opts: CleanKeyOptions = ...

    Returns string
